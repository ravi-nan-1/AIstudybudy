# **App Name**: AI Study Buddy

## Core Features:

- Chat and Learn: Interactive chat interface for Q&A with the AI tutor, powered by google/flan-t5-base and utilizing RAG and vector data for learning, providing detailed explanations.
- Content Upload: Upload content via web URL, PDF, YouTube video, audio, or video files. Automatic content processing for learning and vector data creation.
- Content Library: Display uploaded content for easy access and management.
- Content Summary: Generate summaries of uploaded content using AI to highlight key information. AI decides what to include, reasoning using a tool to choose important facts. Uses RAG and vector data.
- Practice Quiz Generation: Generate quizzes based on selected content with options for 10, 20, or 30 questions. Online exam format with timer and score tracking. AI decides what to include, reasoning using a tool to generate questions from the content. Uses RAG and vector data.
- Model Integration: Automatically downloads and integrates the google/flan-t5-base model for AI tutoring functionality. Uses RAG and vector data to learn.
- Detailed Learning Logs: Generate and display detailed logs of learned information to the student, leveraging RAG and vector data insights.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire trust and focus, and reinforce a feeling of intelligence.
- Background color: Light gray (#F5F5F5) to create a clean, distraction-free learning environment.
- Accent color: Vibrant orange (#FF9800) for interactive elements and CTAs to add some energy and interest.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern and neutral feel, providing great readability for long study sessions.
- Use simple, clear icons to represent different content types and actions.
- Clean, organized layout with clear sections for chat, content library, and quizzes. Use of cards for content display.
- Subtle animations for feedback and loading states to improve user experience.
